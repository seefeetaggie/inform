A work of interactive fiction.

Marine Biologist Gameplay was created with Inform and has IFID E7262AAD-721D-4A77-B159-58CCF93BB702. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. The postcard was written by Andrew Plotkin and designed by Lea Albaugh. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
